# Manual del Conjurador — Avanzado (v1.1)

## 0) Propósito
Operar sesiones M1 con cálculo 𝒱, diseño de gates y cierres reproducibles fieles a Qel.

## 1) Plantillas rápidas
- **FS (YAML)**, **Ficha 𝒱**, **Ficha VF**, **Bitácora** y **Rumbos activos**.

## 2) 𝒱 paso a paso
1) Triada(p) → 2) Objeto(O) → 3) Afinidad A (rúbrica MFH) → 4) χ_r (rumbo) →  
5) H_k (clase/rareza) → 6) Π_gates → 7) 𝒱 clip₀¹ → 8) **V_final** y cierre.

## 3) Gates (cuándo y cómo)
Mediación luminosa/acústica; Doble Testigo; límites de intensidad/tiempo; gate de **Contacto de Aurora** (si aplica).

## 4) Árbol de decisiones
Ética (No-Mentira) → ΔC/ΔS → 𝒱 → Testigos → Veredicto (Cristaliza/Germina/Reposa).

## 5) Herramientas de sesión (CLI)
- `scripts/qel_vcalc.sh` → calcula 𝒱 por objeto.  
- `scripts/qel_session_finalize.sh` → cierra y publica (Diario, MicroSello, VF.PRIMA, ListadoR, manifest).

## 6) Cierre de sesión
SIL → UM → Ə; registrar ΔC/ΔS, V_final, testigos y próximo paso.
